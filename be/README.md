# Braincrackers backend

## Requirements

```txt
python3@3.14.4
uvicorn
tesseract
```

...and python libraries:

```sh
pip3 install -r ./requirements.txt
```

## Run

```sh
uvicorn src.main:app
```

The api is hosted at [http://localhost:8000](http://localhost:8000). See
swagger docs at [http://localhost:8000/docs](http://localhost:8000/docs).
